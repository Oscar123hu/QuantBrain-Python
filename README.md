# QuantBrain-Python

基于 Pandas 与 NumPy 向量化（Vectorized）架构开发的高性能离线策略回测与统计评估平台。系统专注于海量历史行情数据的清洗规整、多维统计特征衍生以及量化核心风控指标的并发计算。

## 核心设计特性

### 1. 全链路向量化计算 (Vectorized Execution)
* **消除显式循环**：在数据预处理与信号激发阶段，全面摒弃 `for` 或 `while` 等高开销循环，全量采用 Pandas 矩阵向量化运算。
* **低延迟回放**：充分利用底层内存连续性与 C 语言级并发加速，将千级别 K 线数据的指标计算与信号回放延迟压缩至毫秒级。

### 2. 数理统计特征工程 (Feature Engineering)
* **置信区间建模**：手写实现布林带（Bollinger Bands）动态高维特征矩阵。通过滚动窗口（Rolling Window）计算 20 周期均值与标准差（$\sigma$）。基于正态分布的 95.4% 统计置信区间边界，向量化捕捉市场超买与超卖状态。
* **时序对齐防漏**：在收益率计算中采用后移一期（`shift(1)`）算法，严格对齐时序撮合逻辑，规避量化回测中的“未来函数”数据泄漏陷阱。

### 3. 风控指标矩阵 (Risk Analytics)
* **最大回撤评级**：利用 `cumprod`（累乘）与 `cummax`（滚动最大值）向量化组合，在 $O(1)$ 空间复杂度内精准捕获历史极限最大回撤（Max Drawdown）。
* **夏普比率测算**：严谨对齐年化收益率与年化波动率，以 2% 无风险利率为基准，输出统计学夏普比率（Sharpe Ratio），客观评估策略的风险收益比。

## 模块职责边界

* `crypto_market_data.csv`：本地结构化历史行情账本（自动同步）。
* `backtest_analyzer.py`：系统核心总控流。集成行情同步、向量化特征工程、交易信号激发以及终极风控评估报告。
